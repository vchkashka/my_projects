## Проекты, выполненные в рамках курса

Данные проекты были выполнены в ходе обучения в Яндекс.Практикуме, профессии "Аналитик данных".

| Название проекта | Описание | Используемые библиотеки | 
| :---------------------- | :---------------------- | :---------------------- |
| [Рынок видеоигр](https://github.com/vchkashka/the_video_game_market) | Необходимо сделать предобработку данных, чтобы команда игры "Секреты Темнолесья" смогла изучить развитие игровой индустрии в начале XXI века. Команда хочет выпустить статью на одном известном IT-ресурсе, которая должна привлечь внимание людей, любящих старые игры, и заодно заинтересовать их «Секретами Темнолесья».| *pandas* |
